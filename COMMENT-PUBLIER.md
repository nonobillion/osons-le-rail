# Publier le site « Osons le Rail ! » sur Vercel

Ce dossier contient un site **statique** prêt à mettre en ligne. Aucun outil technique à installer.

## Contenu du dossier
- `index.html` — le site complet (toutes les pages dans un seul fichier)
- `carte-ligne.png` — la carte ferroviaire affichée sur la page « Notre projet »

> ⚠️ Gardez ces deux fichiers **ensemble dans le même dossier**.

---

## Méthode la plus simple : glisser-déposer (sans compte technique)

1. Allez sur **https://vercel.com** et créez un compte gratuit (avec votre email ou un compte Google/GitHub).
2. Une fois connecté, ouvrez **https://vercel.com/new** puis cliquez sur l'onglet **« Deploy »** / la zone *« Upload »* (ou allez directement sur **https://vercel.com/new/upload**).
3. **Glissez-déposez le dossier `osons-le-rail-site`** entier dans la zone d'upload (ou sélectionnez les fichiers `index.html` + `carte-ligne.png`).
4. Cliquez sur **Deploy**. En ~20 secondes, le site est en ligne avec une adresse du type `osons-le-rail-site.vercel.app`.
5. Cliquez sur **Visit** pour voir le site live. Vous pouvez partager ce lien immédiatement.

## Pour personnaliser l'adresse (optionnel)
- Dans le projet sur Vercel → **Settings → Domains**, vous pouvez ajouter un nom de domaine (ex. `osonslerail.fr`) si l'association en achète un.

## Pour modifier le site plus tard
- Modifiez `index.html`, puis refaites un glisser-déposer sur Vercel (nouveau déploiement). Vercel garde l'historique.
